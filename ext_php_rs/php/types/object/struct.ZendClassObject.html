Allocates a new object when an instance of the class is created in the PHP world.

Internal function. The end user functions are generated by the ZendObjectHandler derive macro which generates a function that wraps this function to be exported to C.

Parameters

  • ce - The class entry that was created.
  • handlers - A pointer to the object handlers for the class.

Safety

This function is an internal function which is only called from code which is derived using the ZendObjectHandler derive macro. PHP will guarantee that any pointers given to this function will be valid, therefore we can Unwrap them with safety.
